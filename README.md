# Stock Price Prediction using Machine Learning

## Objective
Predict the next day's closing stock price using historical stock data and Machine Learning models.

## Dataset
Stock data fetched from Yahoo Finance using yfinance library.

## Features Used
- Open
- High
- Low
- Volume

## Target
- Next day's Close price

## Models Used
- Linear Regression
- Random Forest Regression

## Steps
1. Fetch stock data using yfinance
2. Prepare dataset
3. Train ML model
4. Predict stock prices
5. Compare predictions with actual values
6. Visualize results

## Results
Random Forest provided more accurate predictions compared to Linear Regression.

## Technologies Used
- Python
- Google Colab
- yfinance API
- scikit-learn
- pandas
- matplotlib
